ПРОБЛЕМА: Папка открывается и сразу закрывается
───────────────────────────────────────────────────────────────

Это происходит, если вы кликаете на .bat файл вместо папки!


РЕШЕНИЕ 1: Открыть через адресную строку
───────────────────────────────────────────────────────────────
1. Нажмите Win + E (откроется проводник)
2. В адресной строке вверху введите:
   C:\Users\boiar\tower-bot-telegram
3. Нажмите Enter
4. Папка откроется и останется открытой!


РЕШЕНИЕ 2: Через поиск Windows
───────────────────────────────────────────────────────────────
1. Нажмите Win + S
2. Введите: tower-bot-telegram
3. Нажмите на найденную папку
4. Папка откроется!


РЕШЕНИЕ 3: Через контекстное меню
───────────────────────────────────────────────────────────────
1. Найдите папку tower-bot-telegram в C:\Users\boiar
2. ПРАВОЙ кнопкой мыши на папку
3. Выберите "Открыть" (НЕ "Открыть в новом окне")
4. Папка откроется!


РЕШЕНИЕ 4: Через PowerShell
───────────────────────────────────────────────────────────────
1. Нажмите Win + X
2. Выберите "Windows PowerShell"
3. Введите:
   explorer C:\Users\boiar\tower-bot-telegram
4. Нажмите Enter
5. Папка откроется!


ВАЖНО: Не кликайте на файлы .bat!
───────────────────────────────────────────────────────────────
Файлы с расширением .bat - это скрипты, они запускаются и закрываются.
Кликайте именно на ПАПКУ, а не на файлы внутри неё!
